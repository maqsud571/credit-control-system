# 📘 Credit Control System

**This project is a credit management system written in Python that implements functions such as setting credit limits, granting/returning loans, and managing user permissions using OOP (Object-Oriented Programming).**

# 🧾📌General description
* **Credit Control System** — bu kreditlar bilan ishlash jarayonini boshqarish uchun yaratilgan CLI (komandali qator interfeysi) orqali ishlaydigan dastur. Loyihada quyidagilar mavjud:
* Kreditorlar va mijozlar uchun kredit profil (limit, balans)
* Kredit so‘rovlarini qabul qilish va qayta ishlash
* Kredit berish / to‘lash funktsiyalari
* Foydalanuvchi ruxsatlarini nazorat qilish
* Ob’ektga yo‘naltirilgan strukturaga asoslangan modullar
